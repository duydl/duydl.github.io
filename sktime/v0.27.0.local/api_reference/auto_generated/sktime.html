sktime.

show_versions()[source]#

Print python version, OS version, sktime version, selected dependency versions.

Pretty prints:

  • python version of environment

  • python executable location

  • OS version

  • list of import name and version number for selected python dependencies

Developer note: Python version/executable and OS version are from _get_sys_info Package versions are retrieved by _get_deps_info Selected dependencies are as in the DEFAULT_DEPS_TO_SHOW variable
