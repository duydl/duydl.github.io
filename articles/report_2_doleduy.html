def mul_mat_vec(A, x):
    rs = len(A)
    cs = len(A[0])
    y = [0]*rs
    for i in range(rs):
        for j in range(cs):
            y[i] += A[i][j] * x[j]
    return y

Task 2.1¶

Task 2.1.1¶

This task requires to solve the linear system $\boldsymbol{A}\cdot \boldsymbol{x} = \boldsymbol{b}$ using LU decomposition. The first step in this process is using Crout's Method to find the $L$ and $U$ matrices such that $\boldsymbol{A} = \boldsymbol{L} \boldsymbol{U}$, $\boldsymbol{L}$ is lower triangular and $\boldsymbol{U}$ is unit upper triangular. The defining equations for Crout's Method are $$ l_{i j}=a_{i j}-\sum_{k=1}^{i-1} l_{i k} u_{k j}, \text { where } i \geq j $$ and $$ u_{i j}=\frac{a_{i j}-\sum_{k=1}^{i-1} l_{i k} u_{k j}}{l_{i i}}, \text { where } i<j $$

Then:

$$ \begin{gathered} \boldsymbol{A} \cdot \boldsymbol{x}=(\boldsymbol{L} \cdot \boldsymbol{U}) \boldsymbol{x}=\boldsymbol{L} \cdot(\boldsymbol{U} \boldsymbol{x})=\boldsymbol{b} \\ \boldsymbol{L} \cdot \boldsymbol{y}=\boldsymbol{b} \\ \boldsymbol{U} \cdot \boldsymbol{x}=\boldsymbol{y} \end{gathered} $$

We will forward substitution to solve for $\boldsymbol{Ux}$ and then backward substitution $\boldsymbol{x}$. The functions are defined as follow:

In [530]:
def crout(A):
    n = len(A)
    L = [[0.0]*n for i in range(n)]
    U = [[0.0]*n for i in range(n)]


    for i in range(n):
        U[i][i] = 1.0
        
        for j in range(i, n):
            sum0 = sum(L[j][k] * U[k][i] for k in range(i))

            L[j][i] = A[j][i] - sum0
        for j in range(i, n):
            sum1 = sum(L[i][k] * U[k][j] for k in range(i))
            
            U[i][j] = (A[i][j] - sum1) / L[i][i]
    return L, U

def forward_subs(L, b):
    rs = len(L)
    x=[0]*rs
    for i in range(len(b)):
        x[i] = b[i]
        for j in range(i):
            x[i] -= (L[i][j]*x[j])
        x[i]=x[i]/L[i][i]
    return x 

def back_subs(U, Ux):
    rs = len(U)
    x=[0]*rs
    for i in range(rs,0,-1):
      x[i-1]=(Ux[i-1]-dot(U[i-1][i:],x[i:]))/U[i-1][i-1]

    return x

def LU_sys_solver(L, U, b):
    Ux = forward_subs(L, b)
    x = back_subs(U, Ux)
    return x

Task 2.1.2¶

This task requires to use iterative methods to solve the linear system. The three iterative methods to be implement are Jacobi Method, Gauss–Seidel Method, and SOR(successive over relaxation) Method.

Jacobi Method

In the Jacobi Method, the solution is obtained iteratively via $$ \boldsymbol{x}^{(k+1)}=D^{-1}\left(\boldsymbol{b}-(L+U) \boldsymbol{x}^{(k)}\right) . $$

We could rewrite it as an element-based formula for each row $i$: $$ x_i^{(k+1)}=\frac{1}{a_{i i}}\left(b_i-\sum_{j \neq i} a_{i j} x_j^{(k)}\right), \quad i=1,2, \ldots, n . $$

We implemented the The Jacobi Method and applied it to solve the task as follow:

In [532]:
def jacobi(A, b, alpha, EPS = 1e-10, N = 10000, verbose = True):
    # Initialize matrices, and vectors
    rs = len(A)
    D = [0.0] * rs 
    R = copy.deepcopy(A)
    for i in range(rs):
        D[i] = A[i][i]
        R[i][i] = 0.0
    x = [0.0] * rs
    x_temp = [0.0] * rs
    
    res_norm_lst = [] # Residal Norm List for Plot
    error_list = [] # Error List for Plot
    for k in range(N):
        for i in range(rs):
            x_temp[i] = (b[i] - dot(R[i], x)) / D[i]
        x = x_temp
        res_norm_lst.append(res_norm(b, mul_mat_vec(A, x)))
        error_list.append(res_norm(alpha, x))
        if res_norm_lst[-1] < EPS:
            break
    if verbose:
        print("Iter count: ", k+1)
        print("Solution: ", x)
        plt.plot(range(k+1), res_norm_lst, label = "Residal Norm")
        plt.plot(range(k+1), error_list, label = "Error Norm")
        plt.yscale("log")
        plt.legend(loc='best')
        plt.title("Jacobi Method")
        plt.show()
    return x

SOR Method

Similarly, the defining element-based equation for SOR Method is shown below:

$$x_i^{(k+1)}=(1-\omega) x_i^{(k)}+\frac{\omega}{a_{i i}}\left(b_i-\sum_{j<i} a_{i j} x_j^{(k+1)}-\sum_{j>i} a_{i j} x_j^{(k)}\right), \quad i=1,2, \ldots, n$$

We choose the value of relxation coefficient to be $w = 1.5$ and implement it as follow:

def sor(A, b, w, alpha, EPS = 1e-10, N = 10000, verbose = True):
    # Initialize matrices, and vectors
    rs = len(A)
    D = [0.0] * rs 
    R = copy.deepcopy(A)
    for i in range(rs):
        D[i] = A[i][i]
        R[i][i] = 0.0
    x = [0.0] * rs
    x_temp = [0.0] * rs
    
    res_norm_lst = [] # Residal Norm List for Plot
    error_list = [] # Error List for Plot
    for k in range(N):
        for i in range(rs):
            x[i] = w * (b[i] - dot(R[i], x)) / D[i] + (1-w) * x[i]

        error_list.append(res_norm(alpha, x))

            break
    if verbose:
        print("Iter count: ", k+1)
        print("Solution: ", x)
        plt.plot(range(k+1), res_norm_lst, label = "Residal Norm")
        plt.plot(range(k+1), error_list, label = "Error Norm")
        plt.yscale("log")
        plt.legend(loc='best')
        plt.title("SOR Method")
        plt.show()
    return x

Conjugate gradient method

The conjugate gradient method is an algorithm for solving $\boldsymbol{A x}=\boldsymbol{b}$ where $\boldsymbol{A}$ is a real, symmetric, positive-definite matrix. We input an approximate initial solution vector $\boldsymbol{x}_0$.

$$ \begin{aligned} & \boldsymbol{r}_0:=\boldsymbol{b}-\boldsymbol{A} \boldsymbol{x}_0 \\ & \boldsymbol{p}_0:=\boldsymbol{r}_0 \\ & k:=0 \\ & \text { repeat } \\ & \qquad \alpha_k:=\frac{\boldsymbol{r}_k^{\top} \boldsymbol{r}_k}{\boldsymbol{p}_k^{\top} \boldsymbol{A} \boldsymbol{p}_k} \\ & \qquad \boldsymbol{x}_{k+1}:=\boldsymbol{x}_k+\alpha_k \boldsymbol{p}_k \\ & \qquad \boldsymbol{r}_{k+1}:=\boldsymbol{r}_k-\alpha_k \boldsymbol{A} \boldsymbol{p}_k \\ &\qquad \text{if } \boldsymbol{r}_{k+1} \text{is sufficiently small, then exit loop} \\ & \qquad \beta_k:=\frac{\boldsymbol{r}_{k+1}^{\top} \boldsymbol{r}_{k+1}}{\boldsymbol{r}_k^{\top} \boldsymbol{r}_k} \\ & \qquad \boldsymbol{p}_{k+1}:=\boldsymbol{r}_{k+1}+\beta_k \boldsymbol{p}_k \\ & \qquad k:=k+1 \\ & \text{end repeat} \end{aligned} $$

The algorithm is implemented as the subroutine linear_cg. We confirm the correctness of the subroutine by solving for the solution of $\boldsymbol{Ax_1} = \boldsymbol{b_1}$.

In [535]:
def linear_cg(A, b, e = 1e-10, N = 1000):
    n = len(A)
    xk = [0.0] * n
    rk = [mul_mat_vec(A, xk)[i] - b[i] for i in range(n)]
    pk = [-r for r in rk]
    
    num_iter = 0
    x = [xk]
    for iter in range(N):
        apk = mul_mat_vec(A, pk)
        rkrk = dot(rk, rk)
        
        alpha = rkrk / dot(pk, apk)
        xk = [xk[i] + alpha * pk[i] for i in range(n)]
        rk = [rk[i] + alpha * apk[i] for i in range(n)]
        beta = dot(rk, rk) / rkrk
        pk = [-rk[i] + beta * pk[i] for i in range(n)]
        
        num_iter += 1
        x.append(xk)
        print('Iteration: {}; x = {}; residual = {}'.format(num_iter, xk, norm(rk)))
        if norm(rk) < e:
            break

    return x

Task 2.3.1¶

Consider an $n \times n$ matrix A that has $n$ linearly independent real eigenvalues such that $|\lambda_1| \geq |\lambda_2| \geq \dots \geq |\lambda_n|$ and corresponding eigenvector $v_i$.

Let $x_0$ be an arbitrary vector with $n$ elements, we could write:

$$\begin{aligned} Ax_0 &= c_1Av_1+c_2Av_2+\dots+c_nAv_n \\ &= c_1\lambda_1v_1+c_2\lambda_2v_2+\dots+c_n\lambda_nv_n \\ &= c_1\lambda_1[v_1+\frac{c_2}{c_1}\frac{\lambda_2}{\lambda_1}v_2+\dots+\frac{c_n}{c_1}\frac{\lambda_n}{\lambda_1}v_n]= c_1\lambda_1x_1 \end{aligned}$$

And, similarly: $$\begin{aligned} Ax_1 &= \lambda_1{v_1}+\frac{c_2}{c_1}\frac{\lambda_2^2}{\lambda_1}v_2+\dots+\frac{c_n}{c_1}\frac{\lambda_n^2}{\lambda_1}v_n \\ &= \lambda_1[v_1+\frac{c_2}{c_1}\frac{\lambda_2^2}{\lambda_1^2}v_2+\dots+\frac{c_n}{c_1}\frac{\lambda_n^2}{\lambda_1^2}v_n] = \lambda_1x_2 \end{aligned}$$

As $\lambda_1$ is the largest eigenvalue, therefore, the ratio $\frac{\lambda_i}{\lambda_1}<1$ for all $i>1$. Thus when we repeat multiplying $A$ for sufficiently large amount of times, all the terms that contain $\frac{\lambda_i}{\lambda_1}$ will approach zero. Then, the equation could be simplified to be:

$$ Ax_{k} = {\lambda_1}v_1 $$

To avoid large value, we also need to apply normalization after each iteration, which is usually done by factoring out the largest element in the vector, resulting in the largest element in the vector becoming 1. As a result, after convergence, the factoring out number will equal the largest eigenvalue and the resulting vector is the corresponding eigenvector.

We will set the stopping condition to assess convergence to be the norm of the residual vector being sufficiently small.

Based on the explaination above, we will implement the power method as follow:

In [544]:
def power(A, N = 1000, e = 1e-10):
    x = [1.0]*len(A)
    m = 0
    for iter in range(N):
        x = mul_mat_vec(A, x)
        m_old = m
        m = max([comp for comp in x], key=abs)
        x = [comp * 1.0/m for comp in x]
        
        if abs(m_old - m) < e:
            break
        
    return m, x, iter

Because the eigenvalues of $A^{-1}$ are reciprocals of the eigenvalues of $A$, we can find the smallest eigenvalue of $A$ by finding the largest eigenvalue $\lambda_{max} ' = 1/{\lambda_{min}}$ of $A^{-1}$. Thus, in the inverse power method, instead of multiplying $A$ as implemented in the power method, we will repeatingly multiply $A^{-1}$.

$$x^{k+1} = A^{-1} x^k \iff Ax^{k+1} = x^k$$

As a replacement to multiple the inverse matrix, we will solve the linear equation $Ax^{k+1} = x^k$. The inverse power method is implemented and applied as follow:

Task 2.3.3¶

In a iteration k of the QR method to find eigenvalues of a matrix $A$, we first use Gram Schmidt to find the QR decomposition of matrix $A_k$, getting $Q_k$ - an orthogonal matrix and $R_k$ - an upper triangular matrix. Then we fine the matrix $A_{k+1} = R_kQ_k$ and continue to the next iteration. As k increases, the matrix $A_i$ will finally converge to an upper triangular matr form, the diagonal entries of such matrix will also be its eigenvalues.

However, as $$A_{k+1} = R_kQ_k = Q^{-1}_kQ_kR_kQ_k = Q^{-1}_kA_kQ_k$$ $A_k$ and $A_{k+1}$, and consequently all $A_i$ are similar and have similar eigenvalues.

We will implement Gram Schmidt Method to find the QR decomposition of A and find all eigenvalues of A. In this section we will the class numpy.array for faster acces to the column vector.

In [548]:
import numpy as np
def gram_schmidt(A): 

    n, m = A.shape # get the shape of A

    Q = np.array([[0.0] * n for i in range(n)]) # initialize matrix Q
    u = np.array([[0.0] * n for i in range(n)]) # initialize matrix u

    u[:, 0] = A[:, 0]
    Q[:, 0] = u[:, 0] / norm(u[:, 0])

    for i in range(1, n):

        u[:, i] = A[:, i]
        for j in range(i):
            u[:, i] -= dot(A[:, i], Q[:, j]) * Q[:, j] # get each u vector

        Q[:, i] = u[:, i] / norm(u[:, i]) # compute each e vetor

    R = np.zeros((n, m))
    for i in range(n):
        for j in range(i, m):
            R[i, j] = dot(A[:, j] , Q[:, i])

    return Q, R
